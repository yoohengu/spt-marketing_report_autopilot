# Skill: /review — 최종 검토 (자가 채점)

/generate 완료 후 실행. 리포트 완성도와 수치 정확성을 점검합니다.
모든 항목을 순서대로 확인하고 각 항목 옆에 ✅ 또는 ❌로 표시하세요.

---

## 🟢 Basic 필수 항목 (100점)

**[ ] 1. 핵심 수치 요약 (30점)**
- 총지출·총매출·전체 ROI·총전환수 + 채널별 합계 표 존재
- 5개 채널 모두 포함 (오가닉은 "광고비 0 / ROI 측정 불가" 명시)
- 수치가 Python 계산값인가 (수기 추정 금지)

**[ ] 2. 채널별 ROI 비교 (30점)**
- ROI 순위표 존재 (높은 순), ROI = 매출/광고비×100
- 오가닉(광고비 0)은 순위 별도 표기, 순위 해석 1~2줄 포함

**[ ] 3. 이슈 3가지 (30점)** — 각 이슈에 3요소 모두:
- 현상(무슨 일) / 근거 수치(구체적 숫자) / 권장 조치(대응 방법)
- 후보: 메타 W5 광고비 급등→ROI 급락 / 카카오 W4 급락 / 이메일 매출 이상치·중복·결측 / 오가닉 무비용 고전환

**[ ] 4. 전주 대비 변화율 (10점)**
- 최근 2주(W7→W8) 지출·매출·전환수·CTR 증감율(%) 표 존재

**[ ] 5. 리포트 파일 존재**
```bash
ls -la output/insight_report.md
```

## 🟡 Standard 가산 항목 (+30점)

**[ ] 6. 파이프라인 재현성 (15점)**
- 새 CSV를 넣어도 동일 리포트가 나오는 /명령 또는 Python 스크립트 존재
- 재실행 시 정제→계산→해석→리포트가 자동으로 도는가

**[ ] 7. 주간 리포트 설계 (15점)**
- 섹션·지표·형식을 본인이 직접 기획 + 마케팅팀·경영진 관점 근거 설명

---

## 🔴 Challenge 가산 항목 (+30점)

**[ ] 8. 예산 재배분 기획안 실효성 (20점)**
- 채널 ROI·이슈 근거로 예산 이동 제안 1~2건 (문제 정의·근거 데이터·재배분안·우선순위)

**[ ] 9. 양식·우선순위 설계 (10점)**
- 기획안 양식·우선순위 판단 기준을 본인이 설계 + 근거 설명

## 감점 항목 확인

| 감점 사유 | 확인 방법 |
|---------|---------|
| 수치를 Claude가 직접 계산 | decisions.md에 Python 사용 명시 없음 / "약 X" 표현 |
| 이슈 근거 수치 없음 | "높다/낮다" 등 정성적 표현만 |
| 중복 미제거 | 280행 그대로 합산 (매출·전환 부풀려짐) |
| 이상치 미처리 | 이메일 2026-04-26 매출 포함된 채 집계 |
| 오가닉 ROI 무한대/오류 | spend=0 채널 ROI 계산에서 예외 처리 안 함 |
| 차트 이미지 제출 | 본 과제는 텍스트 리포트만 (시각화 제외 범위) |
