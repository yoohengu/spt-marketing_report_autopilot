4. 예산 재배분 기획안

우선순위 판단 기준

우선순위 점수 = (ROI 개선폭 또는 대체 효율지표) × (재배분 금액 규모, 백만원) ÷ (실행 난이도, 1~3)
  • ROI 개선폭: 돈을 옮겼을 때 가중평균 목표 ROI가 원래 채널 ROI보다 몇 %p 개선되는가. 오가닉처럼 ROI가 정의되지 않는 채널이 대상이면 CVR 격차 같은 대체 효율지표를 사용.
  • 재배분 금액 규모: 실제로 움직이는 돈이 클수록(백만원 단위) 비즈니스 임팩트가 크므로 가중치로 곱함.
  • 실행 난이도(1~3): 1=예산 티어 조정만으로 즉시 실행, 2=담당자 협의·소재 준비 필요, 3=신규 조직·프로세스 구축 필요.
  • 점수 상위 → P0(즉시 실행), 하위 → P1(검토 후 실행).
  • 한계: 기획안 1(ROI 개선폭, 수백 단위)과 기획안 2(CVR 격차, 1대 단위)는 단위가 달라 점수 절대값을 직접 비교하긴 어려움(1673.0 vs 0.37) — 최종 P0/P1은 점수와 함께 "근거의 확실성"도 고려.

P0기획안 1: 메타광고 예산 20% 축소 → 이메일·네이버광고 재배분

문제 정의: 메타광고는 광고비 2위(13,175,534원)를 쓰면서 ROI는 유료 채널 중 최하위(258.7%) — 8주 전체 기준 구조적 비효율
근거 데이터: 채널 ROI 메타 258.7% < 카카오 343.3% < 네이버 658.6% < 이메일 1042.2%. ROAS: 메타 2.59 vs 네이버 6.59 vs 이메일 10.42
재배분 제안: 메타광고 총 광고비의 20%(2,635,107원)를 상시 축소해 이메일·네이버로 재배분 — 배분 비율은 두 채널의 ROAS 비례(이메일 10.42 : 네이버 6.59)로 계산해 이메일 61.3%(1,614,216원) : 네이버 38.7%(1,020,891원). 각 채널 평균 ROAS 선형 추정 시 순증분 매출 약 +16,722,875원, 재배분 후 가중평균 목표 ROI 893.6%(+634.9%p)
우선순위: 점수 1673.0 → P0. 예산 조정만으로 즉시 실행 가능(난이도 1), 이동 대상(이메일·네이버)이 이미 검증된 고ROI 채널
한계: 평균 ROAS가 예산 확대 후에도 유지된다는 선형 가정 — 절반만 4주 시범 적용 후 재측정 권장

P1기획안 2: 카카오광고 예산 15% → 오가닉 콘텐츠/SEO 신규 투자

문제 정의: 오가닉은 광고비 0원인데 매출·전환 전체 2위지만 콘텐츠·SEO 투자가 전혀 없음. 카카오광고는 CVR 1.97%로 오가닉(3.11%)보다 전환 효율 낮음
근거 데이터: 카카오 CVR 1.97% vs 오가닉 CVR 3.11%(격차 1.14%p). 오가닉 8주 평균 WoW 매출 성장률 +1.7%(투자 없이도 자연 성장 중). 카카오 15% 축소 시 손익분기 매출 3,383,513원
재배분 제안: 카카오광고 예산의 15%(986,447원)를 콘텐츠 제작/SEO 신규 예산으로 전환
우선순위: 점수 0.37 → P1. 성과가 즉시 나타나지 않고(난이도 3), ROI 직접 계산 불가(대체지표 추정)라 근거 확실성이 기획안 1보다 낮음
한계: 콘텐츠 투자 효과는 이 데이터만으로 검증 불가 — 발행 전후 오가닉 트래픽을 별도 추적해 손익분기 회수 여부 확인 필요

6. 의사결정 로그
  • 완전 중복 1행(네이버광고 2026-05-15) 제거 → 네이버광고 W7은 6일 기준으로 집계. 임의로 보정하지 않고 그대로 반영
  • 매출 이상치 1건(이메일 2026-04-26, 객단가 196,646원)을 채널별 정상 범위(Tukey's fence) 기준으로 탐지·제거 → 이메일 W4도 6일 기준으로 집계
  • 개별 셀 결측 3건은 0으로 채우지 않고 그대로 두어, 합계 계산 시 해당 값만 자동 제외(다른 지표는 정상 집계)
  • 오가닉 채널은 광고비 0원이라 ROI/ROAS는 "측정 불가"로 표기하고 CTR/CVR로만 성과를 평가
  • 이슈 3가지는 "다음 액션(예산 재배분 등)으로 이어지는 비즈니스 임팩트"를 기준으로 선정 — 데이터 정제 이슈(중복·결측·이상치) 자체는 별도 액션으로 이어지지 않아 이슈 목록에서 제외하고, 5번 데이터 신뢰도 섹션에서 처리 사실만 투명하게 공개
  • 예산 재배분 우선순위는 "ROI 개선폭(또는 대체 효율지표) × 재배분 금액 규모 ÷ 실행 난이도"로 산정 — 오가닉처럼 ROI가 정의되지 않는 채널은 CVR 격차를 대체 지표로 사용

  1. 데이터 정제(중복·이상치·결측 제거) + 채널별 ROI 순위 + 전주 대비 변화율(채널별·전체 합산) 계산 + 이슈 후보 탐지 + 예산 재배분 대상·금액·우선순위 계산 → JSON
  2. JSON을 근거로 리포트(.md) 갱신 후, .md를 그대로 스타일 HTML로 렌더 — .md가 정본이라 두 파일이 어긋나지 않음

참고: 재배분 대상 채널은 매번 실제 ROI·CVR을 기준으로 자동 선정되지만, 축소 비율(20%·15%)은 고정값이다. 더 공격적으로/보수적으로 조정하고 싶다면 src/budget_proposal.py의 비율 설정을 바꾸면 된다.
