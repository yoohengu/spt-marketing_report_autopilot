# [09] DA 공통 — 비즈니스 인사이트 리포트 자동 생성기

트랙: DA | 작업 시간: 3시간 | 개인 과제
채용 TO: 없음 (직무 공통 과제 — 특정 기업 비종속)

## 기업 배경

이 과제에는 특정 채용 기업이 없습니다. **데이터 분석가(DA)가 어느 회사에서나 반복적으로 마주치는 직무 공통 과제**, 즉 마케팅 성과 리포팅 자동화를 다룹니다.

당신은 한 이커머스/서비스 기업의 마케팅 데이터 분석가입니다. 회사는 8주간 5개 채널(네이버광고·메타광고·카카오광고·오가닉·이메일)에 마케팅을 집행했고, 일별 성과 데이터가 CSV로 수집되어 있습니다. 마케팅팀과 경영진은 매주/매월 "어느 채널이 효율적인가, 무엇이 변했는가"를 보고받길 원합니다.

## 현재 상황과 문제

DA는 매주·매월 마케팅팀과 경영진을 위한 성과 리포트를 작성합니다. 그런데 그 과정이 매번 **데이터를 뽑고 → 계산하고 → 시각화하고 → 요약하는 수작업의 반복**입니다.

- **수작업 반복**: 같은 형태의 리포트를 매 주기 손으로 다시 만든다 (반나절 이상 소요)
- **계산 오류 위험**: ROI·변화율·채널 비교를 Excel에서 수동 계산하다 수식·셀 참조 오류 발생
- **인사이트 부재**: 리포트가 "수치 나열"에 그치고, "그래서 무엇을 해야 하는가"를 담지 못함

핵심은 **계산은 정확하게(Python), 해석은 풍부하게(Claude)** 역할을 나누는 것입니다.

## 미션

data/marketing_performance.csv 파일을 분석하여:
1. **핵심 수치 요약**: 총지출·총매출·전체 ROI·총전환수 (+ 채널별 합계)
2. **채널별 ROI 비교**: 5개 채널 ROI 순위표 (ROI = 매출 / 광고비 × 100)
3. **이슈 3가지**: 급증·급락·이상치·결측 중 비즈니스 영향이 큰 3가지를 해석 (근거 수치 + 권장 조치)
4. **전주 대비 변화율**: 최근 2주(W7→W8) 비교 — 지출·매출·전환수·CTR 증감율(%)

⚠️ **합계·ROI·변화율 계산은 반드시 Python으로 처리하세요.** Claude가 직접 계산하면 수식 오류가 발생할 수 있습니다. Claude는 계산이 끝난 수치로 **인사이트 해석**만 담당합니다.

## 입력 데이터

data/marketing_performance.csv — **280행** (8주 W1~W8 × 채널 5개 × 7일)

컬럼(모두 raw 지표 — CTR/CVR/ROAS 같은 계산값은 들어있지 않습니다. 직접 계산이 과제입니다):

| 컬럼 | 설명 | 단위 |
|------|------|------|
| date | 일자 (예 2026-04-01) | 날짜 |
| channel | 채널명 (네이버광고/메타광고/카카오광고/오가닉/이메일) | - |
| impressions | 노출수 | 회 |
| clicks | 클릭수 | 회 |
| spend | 광고비 | 원 |
| conversions | 전환수 | 회 |
| revenue | 매출 | 원 |
| week | 주차 (W1~W8) | 주차 |

데이터 품질 주의사항 (정제가 필요합니다):
- **결측 3건**: impressions 빈 셀 1건, clicks 빈 셀 1건, revenue 빈 셀 1건 (서로 다른 행)
- **완전 중복 행 1건**: 같은 날짜·채널·수치가 두 번 들어간 행 1쌍 (집계 전 제거 필요)
- **이상치 1건**: 특정 일자 이메일 매출이 전환수 대비 비현실적으로 급등 (입력 오류 의심)
- **오가닉 채널 광고비(spend)=0**: ROI = 매출/광고비 계산 시 0으로 나누기 발생 — 별도 처리 필요
- **의도된 주별 급등/급락 패턴**: 일부 주에 광고비·성과가 크게 변동 (이슈·변화율 탐지 연습용)

## 제출 가이드

**기본 (필수)**
- 핵심 수치 + 채널별 ROI 비교 + 이슈 3가지 + 전주 대비 변화율이 담긴 리포트 1개
- output/ 폴더에 저장 (파일명: insight_report.md)

**권장**
- 새 CSV를 넣어도 동일 리포트가 나오는 재현 파이프라인 (/명령 또는 Python 스크립트)
- 주간 리포트 섹션·지표·형식 본인 기획 + 근거

**도전**
- 채널 ROI·이슈 근거로 예산 재배분 기획안 작성
- Skill(/명령)로 패키징하여 재현 가능한 형태로 제출 (또는 구글시트 자동 기입 연동)
