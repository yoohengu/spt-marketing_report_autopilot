## 1. 결과물 임팩트 논의

### Q. output이 약하다는 고민 — 구체화 또는 대안 아이디어?

**현황 문제**: "마케팅 성과 리포트"가 1회성 문서로 끝나면 DA의 AI 활용 역량(반복 자동화·계산 정확성·해석력)이 드러나지 않음.

**핵심 설계 원칙 — 역할 분리**: 합계·ROI·변화율 같은 **계산은 Python(수식 오류 방지)**, 수치 해석·리포트 서술은 **Claude(인사이트)**. 이 분업 자체가 채점 포인트.

| 옵션 | 설명 | 결론 |
|------|------|------|
| A. Python 계산 스크립트 + Claude 해석 | raw CSV → Python 집계 → Claude가 인사이트 작성 | **Basic 핵심**으로 채택 |
| B. Streamlit 대시보드 / 차트 이미지 | 시각화 산출물 | **제외 범위** (텍스트 리포트만) |
| C. Skill 자동화 패키지 | `/analyze→/insight→/generate→/review` 재현 파이프라인 | Standard 레벨로 포함 |
| D. 예산 재배분 기획안 | 채널 ROI·이슈 근거로 예산 이동 제안 | Challenge 레벨로 포함 |

## 2. 레벨 구조 설계

**설계 원칙**: "1회성 리포트" → "매주 돌아가는 자동화" → "데이터→전략 기획"으로 발전하는 누적형(분석가 → PM → 시니어 분석가/PM).

```
🟢 Basic (100점)  — 분석가 수준
  → 280행 raw CSV 정제 → Python 계산 → 핵심수치 + 채널별 ROI 순위 + 이슈 3가지 + 전주대비 변화율
  → 제출: 노션/구글독스 공개 링크 또는 insight_report.md

🟡 Standard (+30점, 누적 130점)  — PM 수준
  → Basic + 새 CSV 넣으면 정제→계산→해석→리포트 자동 생성(/명령 또는 스크립트)
  → 주간 리포트 섹션·지표·형식을 본인이 직접 설계

🔴 Challenge (+30점, 누적 160점)  — 시니어 분석가/PM 수준
  → Standard + 채널 ROI·이슈 근거로 예산 재배분 기획안 (문제정의·근거데이터·재배분안·우선순위)
  → 기획안 양식·우선순위 기준도 본인이 설계
```

## 3. CSV 데이터 재설계 (가장 중요)

### Q. 기존 로컬 파일과 Notion 권위 스펙의 정합성 맞추기

기존 `problem-09-da-insight/data/marketing-performance.csv`는 **35행(7주 × 5채널 주간 집계)** 이었고, CTR/CVR/ROAS가 **이미 계산되어** 들어 있었음. → 학생이 "계산"을 연습할 여지가 없음. Notion 스펙대로 **280행 일별 raw**로 전면 재설계.

**변경 전 컬럼** (구버전, 35행 주간집계 + 계산완료)
```
날짜, 채널, 광고비, 클릭수, 전환수, 매출, CTR, CVR, ROAS
```

**변경 후 컬럼** (Notion v2 기준, 280행 일별 raw)
```
date, channel, impressions, clicks, spend, conversions, revenue, week
```

**주요 변경 사항**
- **행수 35 → 280**: 주간 집계 7주 → **일별 8주(W1~W8) × 5채널 × 7일 = 280행**
- **CTR / CVR / ROAS 컬럼 제거**: 이미 계산된 값 노출 금지 → 학생이 raw에서 직접 계산하는 것이 과제
- **impressions(노출수) 추가**: CTR(=clicks/impressions)을 직접 계산하려면 필수
- **week 컬럼 추가**: W1~W8 (전주대비 변화율·주별 패턴 집계용)
- **날짜**: 주차 라벨 혼용("5월 2주차" 등) → **일별 ISO 날짜**(2026-04-01 ~ 2026-05-26)로 통일
- **채널명 변경**: `네이버검색/카카오/인스타그램/유튜브/이메일` → **`네이버광고/메타광고/카카오광고/오가닉/이메일`** (Notion 스펙). 유튜브·인스타그램을 메타광고로 통합, 오가닉(자연유입) 신설

**채널 성과 차이·트렌드 설계** (현실적 차별화 + 이상패턴 연습)
- 네이버광고: 높은 CVR·ROI, 완만한 주별 성장
- 메타광고: 넓은 노출·낮은 CTR, **W5 광고비 급등(약 3.5배)** → ROI 급락(306%→124%)
- 카카오광고: 중간 성과, **W4 광고비 급락(약 41% 감소)**
- 오가닉: **광고비 0** (ROI 측정 불가 트랩), 전환·매출은 최상위 → 전략 인사이트 소재
- 이메일: 작은 모수·높은 ROAS(약 10배), W6 캠페인 부스트

**의도적 데이터 품질 이슈 (참여자가 처리해야 할 과제)**
- **결측 3건**: impressions 1건(카카오 2026-04-11), clicks 1건(메타 2026-04-20), revenue 1건(네이버 2026-05-07) — 서로 다른 컬럼/행
- **완전 중복 행 1쌍**: 네이버광고 2026-05-15 행이 두 번 존재(W7) — 집계 전 dedup 필요
- **매출 이상치 1건**: 이메일 2026-04-26 매출이 전환수 대비 ~9배(약 3,200만 원) — 입력 오류 의심
- **오가닉 spend=0**: ROI = 매출/광고비에서 0으로 나누기 → 예외 처리 필요(정답 트랩이자 현실 반영)

> 280행은 시드 고정 Python 스크립트로 생성. 최종 CSV 폴더에 존재, 헤더 제외 정확히 280행 검증 완료.
